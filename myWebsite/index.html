Destek Eli
This mobile application was initially developed to enable elderly residents in a nursing home to communicate with each other and with their doctors. Later, it was expanded with additional features such as medication tracking and an emergency button that allows users to quickly contact their relatives in urgent situations.

Multi Stop Route
This project, which I developed during my internship, aimed to determine the shortest route from a mandatory starting location to a set of n possible destinations. The primary goal was to minimize travel time and service duration, thereby improving operational efficiency. Real location data was not used; instead, simulated locations were added for testing and demonstration purposes.

Football Player Scout
Finding the default player filtering system in the football game I played insufficient, I developed a custom filtering interface that allows me to search for players based on specific criteria I define. The goal of this project was to make it easier and faster to find players with the attributes I needed.
